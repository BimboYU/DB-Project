# NGO ERP System - Backend API

## Overview
This is the backend API for the NGO ERP System built with Node.js, Express, and Oracle Database.

## Features
- Complete NGO management system
- Role-based access control (Admin, Staff, Manager, Volunteer, Donor)
- Donation management (monetary & in-kind)
- Campaign tracking
- Volunteer & intern management
- Beneficiary assistance tracking
- Expense management
- Comprehensive reporting and dashboards
- Audit logging

## Prerequisites
1. Node.js 18+ installed
2. Oracle Database 11g/12c/19c/21c
3. Oracle Instant Client installed and configured
